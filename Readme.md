# Design and Analysis of Algorithms - PAT Practice

This repository contains my implementations of classical algorithms practiced in Design and Analysis of Algorithms.

## Algorithms Covered

- Matrix Chain Multiplication
- Lowest Common Subsequence
- N-Queens
- SumOSubset
- Graph Coloring

## Concepts Used

- Backtracking
- Branch and Bound
- Dynamic Programming

## Purpose

This repository is created for:
- Strengthening algorithmic fundamentals
- Practicing DAA concepts

## Main Attraction
Following types of variations of the above algorithms are also added:
- Finding chromatic number in Graph coloring algorithm
- Printing all valid common subsequences in LCS
- Printing the valid contiguous string in LCS
- Maximum Cost in MCM
- Handling duplicates in sum of subsets
- Preplaced queens in N-queens.

Contributions and improvements are welcome!
